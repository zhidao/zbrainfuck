## 【説明】

超難解プログラミング言語Brainfuckのインタプリタzbrainfuckとコンパイラ
zbfcを同梱しました。Brainfuckについては下記ページが参考になります。

http://www.muppetlabs.com/~breadbox/bf/
http://www.kmonos.net/alang/etc/brainfuck.php

--------------------------------------------------------------------
## 【インストール方法】

``
% make
``

で実行ファイル zbrainfuck と zbfc ができます。

--------------------------------------------------------------------
## 【使い方】

どちらも引数にBrainfuckソースファイル名を与えれば実行できます。

``
% zbrainfuck hoge.bf
% zbfc hoge.bf
``

zbfcの出力ファイルはa.outです。

引数を与えなかった場合、または指定されたファイルオープンに失敗した
場合、強制終了します。

言語の硬派(?)な仕様に敬意を表し、エラーメッセージを出力するなどとい
うユーザフレンドリなことは致しません。不正な処理をしたら無言で終了
します。
